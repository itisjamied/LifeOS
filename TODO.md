Long-press task row to open details
Keep the main tap for expanding subtasks, but long-press could jump to habit stats/history. Feels mobile-native.

“Complete all” for a task
If a task has 4 subtasks, maybe the check/plus button on the right can mark all done after confirmation or long-press.

Better empty states
Instead of “Rest day” / “No tasks here,” you could show a small calm card with the day number and maybe “Nothing scheduled.”

Tiny completion animation
When a full task completes: soft pulse, check mark bounce, maybe the row briefly glows. Not too flashy.

Weekly focus view
A compact “This week” strip on Today showing upcoming heavy/light days. Since your cycle matters, this would be genuinely useful.

Skipped / paused state
Sometimes you don’t fail a task, you intentionally skip it. Add “Skip today” so stats can distinguish missed vs skipped.

Custom task icons
Right now symbols are functional, but the whole design is icon-heavy. A better icon picker would make the app feel much more polished.

Reminder nudges
Optional Morning / Evening browser notifications. Keep them opt-in and quiet.


Dark mode contrast pass
You’ve already noticed this theme needs tuning. I’d do one intentional pass over dark mode labels, chips, cards, and colored fills before adding too many features.
